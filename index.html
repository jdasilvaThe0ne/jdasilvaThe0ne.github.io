Robot Operating System ROS2

Let's Learn Robot Operating system(ROS2). This site documents my projects, tutorials, notes, and experiments with ROS2 (including the Kilted Kaju Jazzy build) lets get started.

Before we get started let first answer a few concepts;

  1. Why does industries use ROS

    2. >Robots are becomings widely used in manufacturing industries. Having ROS as opensurce ( a community) allows everyone to use as it, whitout limitation. Persons in the commuity can also Publish to the software. Over the year this has seen to be fruitful to both researchers and manufactuting industries globaly. To get a better understanding of the type of benifits OpenSource ROS has provided check this video on youtube.

  2. Middleware (bridging hardware with applicattion)

    3. In Robtics Middleware acts like a driver. It creates a bridge between the robots hardware coponets suchs as sensors and actuators and the software. Providing a communication pipeline for data exchange and scheduling, which are required for building complex systems(robots). If a robot’s components don’t work in coordination, it stops being intelligent and becomes nothing more than a piece of metal. For a conceptual overview o ROS2 middleware interface please visit ROS2 Middleware interface

  3. Is ROS2 important for real robots

    4. >As mentioned earlier Robots is being largely used in industrial settings, many time diffrern robots work of different aspects of the production. With ROS2's realtime cabbilities, manufacturing plants are provided with resured precise timing. Additionally since each component(Node) of robot works somewhat independenttly but at the same time syncronised, they can be turnred off independently.

  4. Where we use ROS

    5. is openedsource and has been applied in differrent areas within society. Robots for space exploration(robonaut 2 ), customer service(pepper) and manufacruting JetArm Pro. These are some of the areas where the robots built are designed using ROS2
